# RUST learning note
## 1. Installation
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Set default channel, ex. "stable": `rustup default stable`  
Update: `rustup update`  
Uninstall Rust: `rustup self uninstall`
